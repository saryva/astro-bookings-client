# AstroBookings Client Briefing

> ⚠️ Este es el briefing original del ejercicio, sin modificaciones.

## Exercise

Build a client application to interact with the AstroBookings API.
A frontend app to consume the AstroBookings API for offering bookings for rocket launches.

## The client will

* Display available launches with details (rocket, price, seats).
* Allow customers to book seats on a launch, ensuring they do not exceed available seats.
* Show booking status and allow customers to view their bookings.
* Could be implemented as a web or terminal app, using the API to interact with the backend.
  In any language/framework of choice (e.g., React, Vue, Angular, or a CLI tool in Python/Node.js).
  Even a Java or C# console or web app would work.

## Warning

AstroBookings is a fictional space travel company.
The system is designed for demonstration and training purposes.
Not for production use; no security or database is required at the initial stage.
